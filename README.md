# Tasks
Fiz essa atividade pra tentar implementar algum tipo de uso de interfaces, entretando achei desnecesário.
O código só ia aumentar de forma inútil, utilizei alguns métodos de stream com Method reference que aprendi teoricamente ontem, evitei usar lambda porque achei mais simplificado o mtehod reference mesmo.
# Funcionalidades
Esse pequeno projeto permite o usuário criar Tasks, traduzindo, tarefas, excluílas, editá-las e marcá-las como concluídas.
